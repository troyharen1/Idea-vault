# Milestone 1 — Foundation + Deal Intake Skeleton

## Goal

Ship a production-ready skeleton for the first real app so future feature PRs can move fast without changing foundations.

## Deliverables

- Project scaffold (app entry + core module layout)
- Environment variable contract via `.env.example`
- CI + `npm run check` green on pull requests
- Docs for local run and verification
- Clear boundaries for next milestone feature work

## Out of scope (for this PR)

- No deal analysis logic yet
- No external API integrations yet
- No database or auth yet
- No UI yet

## Acceptance criteria

- `npm run check` passes locally and in CI
- App starts with `npm start`
- Repo structure is ready for feature development
